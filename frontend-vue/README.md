# VulnRisk Frontend

Vue 3 frontend application for the VulnRisk platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Dataset management and upload
- Model training and management
- Risk prediction interface
- Interactive reports with charts
